# nemu

nemu is a sleepy tamagotchi made in ruby on rails! :3

The Ruby code backend handles the pet database, evolution, and stats, and a Rails frontend is presented to the user.

Nemu uses Postgres to be fully functional forever and remember your pets.

The style is made to be as similar as possible to a terminal window, but made to be runnable in any browser on any OS (even mobile!)

Deployed on https://nemu.onrender.com :D

**PLEASE NOTE**: For review purposes, the tick speed has been increased from one tick per minute to one tick every 10 seconds. By default, it's six times slower, meaning your nemu is supposed to be able to survive the night, especially if it's sleeping!!
